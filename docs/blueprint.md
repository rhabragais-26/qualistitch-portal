# **App Name**: LeadTracker

## Core Features:

- Customer Data Entry: Allows users to input customer and company name.
- Contact Information: Captures contact number and location details.
- Payment Type Selection: Provides a dropdown to select the payment type: Partially Paid, Fully Paid, and COD.
- Order Specification: Collects data on CSR, order type, and priority type for each lead.
- Product Details: Inputs for product type and source.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) to evoke trust and reliability.
- Background color: Very light grayish-blue (#F0F8FF), nearly white.
- Accent color: Muted grayish-purple (#A090B0) for subtle contrast.
- Font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality, for both headlines and body text.
- Clean, professional icons to represent different data entry fields and actions.
- A clear, organized layout with a focus on easy data input.
- Subtle transitions and feedback animations to enhance user interaction.
